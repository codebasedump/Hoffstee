1. What is a web design?

Web design is the process of developing a website. A web designer is someone who can create a website, but they are not the only ones who can. Web designers use HTML, CSS, and JavaScript to build websites. This process is called coding. There are many different types of web designers, and they use other software to design their websites. Web designers also plan out how the website will look and function before it’s built by using wireframes or prototypes.

2. What do web developer do?

Web designers develop a website’s layout and visual look by choosing colors, fonts, and images to create an appealing site design. Freelancer website developers also develop the user interface, which is what people see when they interact with a website.
